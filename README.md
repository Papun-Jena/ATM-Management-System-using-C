# ATM-Management-System-using-C

A simple ATM Management System developed in C that simulates basic banking operations such as balance inquiry, cash withdrawal, cash deposit, and PIN authentication.

## Features

- Secure PIN Verification
- Three Login Attempts
- Check Account Balance
- Withdraw Cash
- Deposit Cash
- Insufficient Balance Validation
- Amount Validation
- Menu Driven Interface

## Technologies Used

- C Programming
- Functions
- Conditional Statements
- Loops
- Switch Case
